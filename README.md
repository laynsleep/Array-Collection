# Array-Collection
## Array
Sekumpulan deret data yang dapat diakses melalu index

## Collection
Struktur data abstract yang dapat menyimpan dan memanipulasi sebuah objek

## Perbedaan
Perbedaan utama Collection dari Array adalah dari sifat sizenya yang dinamis pada Collection. Selain itu perbedaan lain dari keduanya:

- Heterogen     : Tiap elemen pada Struktur Data Collection bisa memiliki tipe data berbeda
- Abstrak       : Collection memungkinkan untuk mengimplementasikan method custom
- Fleksibel     : Implementasi dapat diganti tanpa merubah cara pakai
